# Contributing to AI Bookmarks

Thank you for your interest in contributing to this project! 🚀

This repository aims to build a curated and structured collection of AI tools to help users discover the best tools efficiently.

---

## ✨ How to Contribute

You can contribute in the following ways:

### 1. Submit a new AI tool
- Provide the tool name
- Include the official website URL
- Add a short and clear description
- Suggest the appropriate category

### 2. Improve existing entries
- Fix incorrect links
- Improve descriptions
- Update outdated or incomplete information

### 3. Suggest new categories
- Help improve the structure of AI tool classification

---

## 📌 Contribution Guidelines

- Keep descriptions concise and informative
- Avoid promotional or overly marketing language
- Ensure the tool is relevant to AI, automation, or productivity
- Prefer widely used or high-quality tools

---

## 🧠 Our Goal

We aim to create a well-structured, easy-to-navigate AI tools directory that is useful for:
- creators
- developers
- founders
- marketers
